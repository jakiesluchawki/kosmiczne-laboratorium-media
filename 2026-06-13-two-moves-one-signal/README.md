# Dwa ruchy, jeden wspólny sygnał

Gotowy pakiet o trybie wspólnej zabawy. Każda osoba odczytuje własną orbitę
i wykonuje własną korektę. Dopiero dwa poprawne ruchy ładują wspólny sygnał
i przesuwają misję dalej.

- `instagram-stories/` - pięć plansz 1080 x 1920;
- `facebook-linkedin/` - dwa obrazy 1200 x 1500;
- `two-moves-one-signal.zip` - pełna paczka do publikacji;
- `linkedin-facebook-post.md` - osobne teksty dla obu kanałów;
- `preview.png` - podgląd całego zestawu;
- `manifest.json` - pochodzenie grafiki i źródła produktowe.
