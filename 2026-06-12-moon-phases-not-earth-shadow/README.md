# Fazy Księżyca: to nie cień Ziemi

Gotowy pakiet o tym, że codzienne fazy Księżyca nie wynikają z cienia Ziemi,
tylko z geometrii układu Słońce-Ziemia-Księżyc i naszego kąta patrzenia.

- `instagram-stories/` - pięć plansz 1080 x 1920;
- `facebook-linkedin/` - dwa obrazy 1200 x 1500;
- `moon-phases-not-earth-shadow.zip` - pełna paczka do publikacji;
- `linkedin-facebook-post.md` - osobne teksty dla obu kanałów;
- `preview.png` - podgląd całego zestawu;
- `manifest.json` - źródła naukowe i pochodzenie grafiki.

Pakiet korzysta z malarskich assetów Kosmicznego Laboratorium i prawdziwego
ekranu laboratorium faz Księżyca z działającej aplikacji.
