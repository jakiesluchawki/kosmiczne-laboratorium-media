# Voyager 2: cztery wielkie planety

Gotowy pakiet o Wielkiej Podróży Voyagera 2 przez Jowisza, Saturna, Urana
i Neptuna.

- `instagram-stories/` - pięć plansz 1080 x 1920;
- `facebook-linkedin/` - dwa obrazy 1200 x 1500;
- `voyager-grand-tour.zip` - pełna paczka do publikacji;
- `linkedin-facebook-post.md` - osobne teksty dla obu kanałów;
- `preview.png` - podgląd całego zestawu;
- `manifest.json` - źródła i parametry publikacji.

Pakiet wykorzystuje malarską mapę planet oraz prawdziwy ekran misji z
Kosmicznego Laboratorium. Tekst wyjaśnia mechanizm kolejnych asyst
grawitacyjnych i zachowuje obraz pełnokadrowy.
