# Pierwsza minuta prowadzi do hipotezy

Gotowy pakiet o początku doświadczenia. Pokazuje, jak otwarcie obserwatorium,
wybór misji i pierwsze pytanie prowadzą dziecko do własnego przewidywania,
zanim ruszy eksperyment.

- `instagram-stories/` - pięć plansz 1080 x 1920;
- `facebook-linkedin/` - dwa obrazy 1200 x 1500;
- `first-minute-to-hypothesis.zip` - pełna paczka do publikacji;
- `linkedin-facebook-post.md` - osobne teksty dla obu kanałów;
- `preview.png` - podgląd całego zestawu;
- `manifest.json` - pochodzenie grafiki i źródła produktowe.
