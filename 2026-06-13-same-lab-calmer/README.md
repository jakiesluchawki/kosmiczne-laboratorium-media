# Spokojniejszy odbiór, to samo laboratorium

Gotowy pakiet o profilu dostępności `Czytaj i upraszczaj`. Materiał pokazuje
większy tekst, mocniejszy kontrast, głos i ograniczenie ozdobnego ruchu bez
usuwania fizyki eksperymentów ani kroków odkrycia.

- `instagram-stories/` - pięć plansz 1080 x 1920;
- `facebook-linkedin/` - dwa obrazy 1200 x 1500;
- `same-lab-calmer.zip` - pełna paczka do publikacji;
- `linkedin-facebook-post.md` - osobne teksty dla obu kanałów;
- `preview.png` - podgląd całego zestawu;
- `manifest.json` - pochodzenie grafiki i źródła produktowe.
