# Orbita to ciągłe spadanie

Gotowy pakiet o tym, dlaczego satelita nie uderza w Ziemię, chociaż
grawitacja cały czas go przyciąga. Sekwencja prowadzi od pytania przez ruch
w bok do prawdziwego ekranu eksperymentu.

- `instagram-stories/` - pięć plansz 1080 x 1920;
- `facebook-linkedin/` - dwa obrazy 1200 x 1500;
- `orbit-is-falling.zip` - pełna paczka do publikacji;
- `linkedin-facebook-post.md` - osobne teksty dla obu kanałów;
- `preview.png` - podgląd całego zestawu;
- `manifest.json` - źródła naukowe i pochodzenie grafiki.
