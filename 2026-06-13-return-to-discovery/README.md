wyjaśnienie nie są zamkniętym wynikiem: dziecko może wrócić do próby,
zobaczyć swój zapis i zaktualizować go po kolejnej obserwacji.

- `preview.png` - podgląd całego zestawu;
- `manifest.json` - pochodzenie grafiki i źródła produktowe.
