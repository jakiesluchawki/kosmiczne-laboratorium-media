# To samo laboratorium na każdym ekranie

Gotowy pakiet o responsywnym układzie aplikacji. Telefon i tablet pokazują ten
sam przepływ uczenia, ale układają go inaczej, żeby pytanie i następny krok
pozostały czytelne.

- `instagram-stories/` - pięć plansz 1080 x 1920;
- `facebook-linkedin/` - dwa obrazy 1200 x 1500;
- `same-lab-every-screen.zip` - pełna paczka do publikacji;
- `linkedin-facebook-post.md` - osobne teksty dla obu kanałów;
- `preview.png` - podgląd całego zestawu;
- `manifest.json` - pochodzenie prawdziwych ekranów i grafiki.
