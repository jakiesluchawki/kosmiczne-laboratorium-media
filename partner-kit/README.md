# Kosmiczne Laboratorium — pakiet współpracy

Modułowy zestaw do pierwszego kontaktu z centrum nauki, muzeum, biblioteką,
domem kultury, fundacją, organizacją społeczną, wydawcą lub partnerem
edukacyjnym.
