# Ten sam moment. Zima w Polsce, lato w Nowej Zelandii

Gotowy pakiet o przeciwnych porach roku na dwóch półkulach. Model nie
przesuwa Ziemi ani Słońca: zmienia się tylko miejsce obserwacji. Dzięki temu
Polska i Nowa Zelandia pokazują różnicę w długości dnia i kącie padania
światła w dokładnie tym samym momencie orbity.

- `instagram-stories/` - pięć plansz 1080 x 1920;
- `facebook-linkedin/` - dwa obrazy 1200 x 1500;
- `one-orbit-opposite-seasons.zip` - pełna paczka do publikacji;
- `linkedin-facebook-post.md` - osobne teksty dla obu kanałów;
- `preview.png` - podgląd całego zestawu;
- `manifest.json` - pochodzenie grafiki i źródła produktowe.
