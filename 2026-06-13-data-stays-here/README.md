# Postęp zostaje na urządzeniu

Gotowy pakiet o prywatności dzieci: bez konta, reklam, zewnętrznej analityki
i profilu budowanego z odpowiedzi. Imiona są opcjonalne, a postęp zostaje
lokalnie w przeglądarce.

- `instagram-stories/` - pięć plansz 1080 x 1920;
- `facebook-linkedin/` - dwa obrazy 1200 x 1500;
- `data-stays-here.zip` - pełna paczka do publikacji;
- `linkedin-facebook-post.md` - osobne teksty dla obu kanałów;
- `preview.png` - podgląd całego zestawu;
- `manifest.json` - pochodzenie grafiki i źródła produktowe.
