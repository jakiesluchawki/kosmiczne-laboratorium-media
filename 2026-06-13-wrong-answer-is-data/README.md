# Zaskoczenie jest częścią eksperymentu

Gotowy pakiet o nietrafionym przewidywaniu. Materiał pokazuje, że aplikacja
zostawia wybór dziecka, podaje konkretny wynik bez punktów ujemnych i pozwala
uruchomić próbę ponownie.

- `instagram-stories/` - pięć plansz 1080 x 1920;
- `facebook-linkedin/` - dwa obrazy 1200 x 1500;
- `wrong-answer-is-data.zip` - pełna paczka do publikacji;
- `linkedin-facebook-post.md` - osobne teksty dla obu kanałów;
- `preview.png` - podgląd całego zestawu;
- `manifest.json` - pochodzenie grafiki i źródła produktowe.
