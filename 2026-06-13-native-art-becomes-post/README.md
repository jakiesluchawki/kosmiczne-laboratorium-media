# Grafiki do postów zaczynają się w aplikacji

Gotowy pakiet o języku wizualnym Kosmicznego Laboratorium. Pokazuje drogę od
wybranego malarskiego kierunku i działającego ekranu do grafiki, która wygląda
jak kolejna strona tego samego świata.

- `instagram-stories/` - pięć plansz 1080 x 1920;
- `facebook-linkedin/` - dwa obrazy 1200 x 1500;
- `native-art-becomes-post.zip` - pełna paczka do publikacji;
- `linkedin-facebook-post.md` - osobne teksty dla obu kanałów;
- `preview.png` - podgląd całego zestawu;
- `manifest.json` - pochodzenie grafiki i źródła produktowe.
