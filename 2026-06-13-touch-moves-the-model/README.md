# Ruch palca zmienia model

Gotowy pakiet o laboratorium Księżyca. Pokazuje, że przeciągnięcie obiektu nie
jest ozdobnym gestem: zmienia położenie Księżyca na orbicie i fazę widoczną
w okularze. Te same fazy można też wybrać dużymi przyciskami.

- `instagram-stories/` - pięć plansz 1080 x 1920;
- `facebook-linkedin/` - dwa obrazy 1200 x 1500;
- `touch-moves-the-model.zip` - pełna paczka do publikacji;
- `linkedin-facebook-post.md` - osobne teksty dla obu kanałów;
- `preview.png` - podgląd całego zestawu;
- `manifest.json` - pochodzenie grafiki i źródła produktowe.
