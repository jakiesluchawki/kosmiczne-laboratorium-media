# Siatka jest modelem, nie zdjęciem kosmosu

Gotowy pakiet o świetlnej siatce w pierwszym eksperymencie. Materiał wyjaśnia,
że siatka pomaga porównać wpływ różnych mas na ruch, ale nie jest dosłowną
tkaniną ani zdjęciem przestrzeni.

- `instagram-stories/` - pięć plansz 1080 x 1920;
- `facebook-linkedin/` - dwa obrazy 1200 x 1500;
- `grid-is-a-map.zip` - pełna paczka do publikacji;
- `linkedin-facebook-post.md` - osobne teksty dla obu kanałów;
- `preview.png` - podgląd całego zestawu;
- `manifest.json` - pochodzenie grafiki i źródła produktowe.
