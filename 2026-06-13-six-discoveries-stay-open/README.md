# Sześć odkryć pozostaje otwartych

Gotowy pakiet o mapie misji. Pokazuje wszystkie sześć odkryć przed rozpoczęciem
pracy, wyróżnia następny sensowny krok i nie blokuje pozostałych kart.
Ukończone odkrycia nadal można otwierać i powtarzać.

- `instagram-stories/` - pięć plansz 1080 x 1920;
- `facebook-linkedin/` - dwa obrazy 1200 x 1500;
- `six-discoveries-stay-open.zip` - pełna paczka do publikacji;
- `linkedin-facebook-post.md` - osobne teksty dla obu kanałów;
- `preview.png` - podgląd całego zestawu;
- `manifest.json` - pochodzenie grafiki i źródła produktowe.
