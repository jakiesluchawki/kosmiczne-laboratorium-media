# Czy magnes przyciąga każdy metal?

Gotowy pakiet o tym, że zwykły magnes nie przyciąga po prostu „metalu”, tylko
silnie reaguje z wybranymi materiałami. Sekwencja porównuje także magnetyzm
z grawitacją, która działa na każdy przedmiot z masą.

- `instagram-stories/` - pięć plansz 1080 x 1920;
- `facebook-linkedin/` - dwa obrazy 1200 x 1500;
- `magnet-not-every-metal.zip` - pełna paczka do publikacji;
- `linkedin-facebook-post.md` - osobne teksty dla obu kanałów;
- `preview.png` - podgląd całego zestawu;
- `manifest.json` - źródła naukowe i pochodzenie grafiki.

Pakiet korzysta wyłącznie z malarskich assetów Kosmicznego Laboratorium i
prawdziwego ekranu laboratorium magnesów z działającej aplikacji.
