# Muzyka może zostać, kiedy głos milczy

Gotowy pakiet o dwóch niezależnych warstwach dźwięku. Narrację można
wyłączyć bez wyciszania orkiestry, a podczas ruchu aplikacja nie opisuje
każdej zmiany. Muzyka buduje atmosferę, ale eksperyment nadal zostawia
miejsce na patrzenie i własną myśl.

- `instagram-stories/` - pięć plansz 1080 x 1920;
- `facebook-linkedin/` - dwa obrazy 1200 x 1500;
- `music-stays-voice-rests.zip` - pełna paczka do publikacji;
- `linkedin-facebook-post.md` - osobne teksty dla obu kanałów;
- `preview.png` - podgląd całego zestawu;
- `manifest.json` - pochodzenie grafiki i źródła produktowe.
